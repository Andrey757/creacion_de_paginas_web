CATALOGO ONLINE

¿Qué es una web catálogo online?
Una web catálogo online, también conocida como web catálogo de productos online, es una web que tiene todas las prestaciones de un sitio web corporativo, pero que añade la posibilidad de albergar información de productos u otros contenidos en forma de fichas, para poder agruparlos en secciones, subsecciones, familias, tipos, etc.

El hecho de poder almacenar la información en fichas también facilita la actualización de las mismas, tanto si se hace de forma manual como si se utiliza el conector de webnet, el cual le permitirá automatizar las tareas de mantenimiento de los datos, ya sea directamente desde su aplicación de gestión, ERP, hojas de cálculo o cualquier otra base de datos.
